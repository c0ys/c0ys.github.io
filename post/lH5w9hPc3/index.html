前言

因为是闲置的笔记本，总想着整些花的来，之前已经经历了不同发行版的Linux，问题依旧是

  • Fuck Nvidia (现在Apple也拒绝合作了，出了名对开源极度不友好。)

  • 常用软件的安装使用（eg:qq）

  • 中文字体显示优化

  • 不稳定，动不动给你死机（身娇体弱易推倒.....）

这些日子分别经历了Kali->Deepin->Hackintosh.

以上情况在Deepin有所改善，但是对于我这样菜鸡来说，Linux还是不适合做独立的个人系统的。作为服务器运行环境或者学习环境是完全OK的。现在我的使用一般是虚拟机，WSL，Centos。

在一个说巧不巧的时机，我又看到了Hackintosh，以前我也安装过，不懂世事的我，在分辨率(HI-DPI)，键盘反人类(Command)，驱动安装(触控板)问题上就给止住了，没有继续用下去。但人是会学习的，这两年也经历了不少，有了更多的理解。

不同于开源的Linux，闭源的黑苹果相对稳定多了，这时我又想起以前知乎的看到别人调侃这三者。

windows是风骚熟女，活好，开放，要玩啥都会陪你玩，而且玩到舒心。但是搞不好爱给你带点东西回来。
mac os x 是傲娇萌妹，漂亮，在一起舒服，拿出去气派。但是你就要听我的，买我的。不然滚，哼！这里不许碰！就不让你碰！去你的，从床上给我滚下去！
linux 是懵懂萝莉，啥都听你的，都干，但是要自己慢慢调教，调教过头还容易把小身子骨搞坏了。。

抱着反正笔记本也是闲置，不如赌一赌单车变摩托🏍的心态又搞了一把。一搞就是一天多，我想着把我愚蠢的经历都写下来。

很多人对黑苹果有争议，部分人觉得就是这群屌丝没钱就瞎折腾。见仁见智把，我的确是为了尝鲜而去体验苹果系统，但更多的是对未知领域的探索和搞机的乐趣。我觉得Hackintosh本身是一种极客精神。是对黑客精神的一种诠释。对于把安装好黑苹果当优越感的人，那他们是真的很失败。我并不觉得这是一件值得炫耀的事。那些对技术有追求，真正需要的人，他们重点是放在Mac给他们能带来多少生产力，和安装Hackintosh过程中能够学习多少知识。我认为这才是Hackintosh真正存在的价值。 途中我会遇到很多问题，我会不断的学习，去google资料，去逛论坛，英语不好的我用翻译看外国佬的论坛和网站，乐此不疲，一个个解决他们，看到正确显示的时候，那便是我最快乐的时候，成就感那必须Max，人嘛，活着不就图一乐，在我看来安装的这个过程远比Hackintosh本身要意义的多。折腾自己喜欢的东西，这个过程本身就是一种享受。

以下叙述按时间线叙述。

初试 macOS Mojave 10.14.5

我当时下的是macOS Catalina 10.15.3 ，这是现在最新的系统了，虽说可能存在未知bug，驱动不完善，但是这怎么能挡得住我一颗炙热的心。

国内论坛交学费

当时还在了解信息，只会去网站找EFI，我信了他的鬼话，有各式各样的EFI，交了学费。还只能下两种版本的EFI，数量，质量还可怜。MDGB，黑心坑人淘宝店，CLGJ.开源精神万岁！幸运的我找到同款的EFI，Github Yes！

Github 优秀资源总结

后面无意中找到了这篇优秀黑苹果总结。让我找到了我同款，类似型号的EFI。

精品总结：https://github.com/daliansky/Hackintosh

首先得感谢这位大佬基本同款的配置，整理好的EFI，比起那些淘宝店粗糙烂制，明显个人对自己的个人电脑还是上心的很多。这位大佬装了Windows 10，Ubuntu，Hackintosh。

笔记本已经不是我的主力机了，自然不用考虑那些头疼一批的多系统引导问题，一个Hackintosh就够了。

它的EFI，的确帮助我许多，起码让我看到了苹果的LOGO，安装了系统，等待了10多分钟，却是后面无法进系统。因为我安装的是最新的macOS Catalina 10.15.3，我仔细看了，原作者是只支持到10.14.5。

This is a Clover install for MacOS Mojave 10.14.5. This build is now a triple boot, Mojave, Windows 10 and Ubuntu build.

经历了上面的折腾，我是怕了，宁可重新下了10.14.5，也先让我装成功再说，果不其然成功进入系统！

10.14.5遇到的超巨大问题—声卡

这个是我遇到的第一道门槛。高的可怕。

其实只要主要最重要的一点：除了AppleALC，把所有关于声卡的驱动都删了就好，这得感谢一个up主，亲身经历并找到问题，然后录下视频，侥幸被我看到。

然而我把所有的注入点都试了一遍，一个都不能用。电脑还是没声音。心力交瘁 +20

然后偶然看到上面up的视频，解决了问题！有声音了！心情愉悦 +10

但是就巧了，玄学，只有耳机有声音，麦克风没声音。我翻了点帖子，爬楼，可能是注入id和自身情况不一样，需要自己仿生声卡驱动。心力交瘁 +40

这一搞可就攀上了高楼。下面只是我其中的一点记录，这个真的是太难了，门槛太高了把。

https://blog.daliansky.net/Use-AppleALC-sound-card-to-drive-the-correct-posture-of-AppleHDA.html

https://github.com/btwise/TOOLS/blob/master/AppleALC/%E5%AE%9A%E5%88%B6AppleALC.md

0x18 24 Mic 外置麦克风 24->12->2->2 2->2->12->24
0x12 18 Internal Mic in 内置麦克风 18->36->7 7->36->18

会不会想问，这他妈是个啥，我也是一头懵逼呀。到了百分之70那里，我是真看不下去他的教程，远超过我的知识理解范围。鬼知道那天我经历了什么。心力交瘁 +100

然后，这并不是励志故事，在大概百分之80的完成度的时候，我实在是看不懂他的说明，两个人各有理解。

但是这个过程我对驱动和efi理解更深了，这个十分重要，影响到我后面的操作。

我放弃了制作仿声卡驱动，理解到一定地步之后，直接推到重来！这次我想直接安装最新的macOS Catalina 10.15.3 ，因为根据我上面的操作原理，我第一步其实是安装成功了10.15的版本，只不过启动的时候驱动失败了！我只要在驱动上面调整一番，应该是能正常启动的。达成成就：大彻大悟！

再试macOS Catalina 10.15.3

这里我进行了思路重新整理，确保驱动只要正常，我应该是有大把握能进10.15.3。

这里多亏了这位用户整理了出来，也是看了他的安装，我才知道，是可以装进最新的10.15.3系统的。

EFI : https://github.com/aminenoidea/Hackintosh-GL63

这里推荐用etcher，不再使用老掉牙的Transmac，界面老，速度还慢。

写入之后，我尝试一步步的替换原有的efi，尝试找出哪一个驱动是能够正确进入10.15.3的方针。

该死的是，我基本每个不同的kext都试了一遍。不知所措 + 20

最后破罐子破摔，直接把上面的EFI，全部复制进去，然后就进去了？？卧槽？？

我承认我有一丝赌的成分，但我赌对了。 进去系统的第一感觉太棒了。

酷~！最新的系统！MacBook Pro！幸福感 +50

优化方面

  • 声卡

  1. 去除万能声卡以及任何声卡驱动—不能用 领悟 +40

  2. 全部id都试了，注入点都GG—不能用 不知所措 +30

  3. 屏蔽相关AppleHDA 要素察觉 +100

  4. 声卡 YES！完美~~

  • 网卡

    • 免驱的BCM94360CS2 成就：必要的投入

结语

难以避免的是，身边还是许多人无法理解我这种行为，他们会怎么想这件事我并在乎。

让我搞到一丝难过的是，好像没有人能理解我的所作所为，感觉自己看起来像个小丑一样自言自语，自乐其中，孤身一人，难免会有些忧伤.

两天后网卡到了！

BCM94360CS2网卡到了，从床上蹦起来赶紧装上。因为苹果的网卡延长了，模具有点紧，但我慢慢蹭蹭，还是进去了，原来还有个防静电的一层膜，撕下来后都不黏了。

使用初体验

谈谈感受把，第一感觉还是mac os的丝滑流畅，这个动画真的甩window和各大手机厂商一截，尽管他并不是最快的，但看起来观感的确很好。丝滑~~

还有我特别不爽的一点，键盘，mac的键位不一样，所以windows下的很多快捷键都不能用，我本来还想改，但还是入乡随俗，慢慢的适应，的确得承认，苹果键盘的设计是有一定道理的，切换输入法和大小写那里，我觉得相比windows来说很明智，以至于我后面都思考了windows的快捷键优化，我把默认语言改成了英文，这样windows10不会总给我弹中文输入法。

快捷键真的是硬伤，两台电脑切换，我手都要抽筋了，切换输入法一个是caps locks，一个是shift；切换大小写一个是shift，一个是caps locks，复制一个是command + c，一个是ctrl + c，我的肌肉记忆不允许我在两者切换自如，按的手都要抽筋了，巨头疼，统一的标准不好么？在MAC下我唯唯诺诺，在Windows我重拳出击。windows下我快捷键飞起，效率很高，毕竟也用了这么多年了，mac下我感觉我和个痴呆一样，有点让我不爽的是

  • 鼠标中键不能关闭网页

    垃圾雷柏鼠标，修了一下，滑轮那不灵了，鼠标问题，陪了我两年，该退休了，过几天让他退休把。

  • 和快速返回桌面（WIN+D） F11，可以全屏相当于返回桌面，但mac好像也不需要返回桌面去做些什么。可能有快捷键把，但是我还不知道，毕竟我才刚玩了几个小时。

安全性，挺过分的，总要输密码，有点小烦，和linux一样，懂得都懂，后期想办法优化一下。

还有就是苹果的生态，自带的app基本一半功能我是用不上的，这让我有点物没有尽其用的感觉，信息，备忘，电话，巴拉巴拉，想必一套苹果的生态那必然效率很高。

smb连接，这个我真得夸，因为在刚上手mac，很多东西需要整理，下载软件，科学冲浪的什么巴拉巴拉，还是没windows我配好的环境块，所以我把windows下的文件共享了，mac直接局域网访问就行。这个我在deepin，kali，ubuntu的linux中，甚至是windows自身，体验都没有mac好，在地址栏输入//192.168.**，直接会弹到访达，哇，这个体验是真的好极了~~

我现在想想，如果要装linux当个人系统，为什么不拿黑苹果做呢，系统界面体验简直是天差地别，大佬别喷，图形界面美化我也干过，多费劲就不说了，崩了，我还不知道咋修。我菜我承认。而且苹果生态里常用的软件也有，简直弥补了我之前的所有需求。

我还是在考虑自己台式要不要也转到苹果生态试试？毕竟还是陌生环境，我还是惧怕不适应，工作环境太过巨大。先慢慢用笔记本玩着吧，反正真的是丝滑，不知道是我配置不够卡了，还是黑苹果的不完善，有的时候总会卡一卡，多数是在苹果相关的应用中，可能也是网速问题把（科学冲浪），苹果官网还是在国外的，存疑。

但是我今天这几个小时折腾下来，我感觉笔记本装的hackintosh是太正确的决定，比装个linux‘稳定’太多了，而且也有用起来的欲望，我没想着用个黑苹果有多装逼，而是真真切切想去了解这个操作系统，不然怎么谈搞机呢？

我可以不用，但我不能不懂。

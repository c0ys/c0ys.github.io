Prestige-P100黑苹果实录

2020-03-11 8 min read

配置 - MSI Prestige P100

显卡原为MSI RX580 ARMOR 2048SP，降级vbios为蓝宝石 RX570用来免驱。

电脑型号： 微星 Z390 Prestige P100
处理器： 英特尔 Core i5-9400F @ 2.90GHz 六核
主板： 微星 MPG Z390I GAMING EDGE AC (MS-7C04) ( Z390 芯片组 )
内存： 16 GB ( 海盗船 DDR4 2400MHz )
主硬盘： HS-SSD-C2000 512GB ( 512 GB / 固态硬盘 )
显卡： AMD Radeon RX 570 ( 8 GB / 蓝宝石 )
声卡： 瑞昱 ALC892 @ 英特尔 High Definition Audio 控制器
网卡： 瑞昱 RTL8168/8111/8112 Gigabit Ethernet Controller / 微星

起因 - 怀疑卡顿原因

笔记本黑苹果体验还是很好的，hidpi开启后，高分辨率的享受让我如痴如醉，但是就一弊端，卡顿，我思考相关的原因。

  • HD630过于弱鸡但是白苹果也是同样，确为何如此丝滑？

  • 黑苹果驱动不完善这个是头疼的，玄学问题，也不知道哪的问题。看我寻思着也不应该呀，我电脑配置不算低把？带着疑惑接着往下走。

后话：其实问题就是HIDPI开启后，显卡过载导致系统缓慢，白苹果原生支持HIDPI，源于自身硬件屏幕素质过硬。 找出这个的原因代价可真是有够大，费了不少心血。

由于这个原因，我十分想探索为何如此，因此我把我的毒手伸向了我的台式—Prestige P100，一台微星设计师系列的主机。

如果台式机能流畅运行(RX580独立显卡硬解)的话？是不是能排除一些问题呢？如果台式机用显卡十分流畅，那就可以确定在显卡的问题上。

不光是为了排除一些问题，电脑摆在这，实在是想折腾一下呀。成就：无味的折腾

豪赌 - 强刷显卡bios

台式机安装黑苹果这个想法我之前就有过，但是存在两大问题。

  • 显卡我的显卡是RX580 2048sp，之前大家都说不能驱动黑苹果，因为这是一张换皮的RX570，现在网上还是很多硬件购买特地声明不要购买这款，不巧，这张卡早在我手里了。这下难搞了把，但是还是有大神冒着试试的态度，刷RX570的vbios，结果成功，黑苹果免驱。这听起来可真的是诱惑人呀！但是刷bios风险巨大，万一刷砖了，我可没东西救，cpu没核显，itx主板没有多余的pcie槽，也就是我刷崩了我就电脑不能用了基本，这注定是一场有去无归的一场赌博。 达成成就：殊死一搏

    其实我已经想好了后路，买个一个核显的奔腾亮机。但是又是一笔支出。成就：完善的安排

  • 无线网卡无线网卡拆装巨大，今天拆机看了，基本要拆下主板才能卸下了，这机子，实在拆不动了，点到为止。还是用usb网卡和usb蓝牙把。itx限制了。

但上面的风险仍然是阻止不了我想刷显卡的心，毕竟高收益可是黑苹果免驱，不用再去买新卡，卖旧卡，现在手里的这张卡就可以免驱，只要你敢试！

因为我知道放着这个可能能免驱的显卡在我这，迟早要被刷bios，逃不过我的，反正迟早要刷，这是一条不归路，成败就在一瞬之间。

也不知道我按下刷下bios的回车键之后我心里多忐忑，又激动又是紧张的。

刚开机的时候，显示器没反应，我一边放着小缘的好运来，一边等着回应，没有反应。但我还是因悲生喜，激动的狂笑，听起来怪矛盾的，因为出问题了，我又能折腾了，又可以把他修好了，去享受修好的成就感。

几番尝试之后，阴差阳错下，我开始察觉不对成就：要素察觉！，我试着重新拔了dp口，换了个口！？成了！！显示器亮了！！！手机里，小缘的好运来还放着，我真的是激动的要跳起来了！！！这是我这几天最激动的事！简直和拿奖一样，这是一场豪赌，我赌对了。我还凭借我敏锐的直觉发现了问题，简直不能太高兴，万一GG了，代价可就大了去。

也许给我一张双bios的显卡，或者多pcie插槽的主板，核显的cpu，其他的电脑等救砖手段，我可能还不会这么激动。因为当我做出这个决定的时候，我已经没有回头路了。也不知道我冒了多大的勇气。

但是还是发现一个小问题，3个dp口，原来插的那个dp口不能用了，其他正常，也无伤大雅把。成就：必要的牺牲

Hackintosh中RX580 2048sp免驱成功！

安装 - 枯燥的等待

台式机中问题还是有点小多，但是后面都莫名其妙的解决了。

  • 保持windows稳定的前提下改硬盘格式 RAID -> AHCI百度解决。安全模式很简单。

  • 安装花屏，卡机，速度缓慢换成rx570之后，原来的efi总是有线条，花屏到不算，但很慢，一看就是显卡驱动那问题，装了很多遍，也是这个过程中认识了oc。这是今天最难熬的部分，因为加载一次很慢，要十几分钟不止，而且还是必须要等跑完才能获取足够信息。换了很多efi，耗费大量时间。而且还是台式机，装机的时候，没有闲置的电脑可以查找资料。这个过程大概持续了2-3个小时，后面我慢慢理清思绪，不能再这么盲目的try下去，把我笔记本安顿好之后，我试着去找efi，又回到那个。黑心网站店，看到一个和我配置很相近的，竟然还要收费，一块钱。哎，算了，像恶势力低头。成就：生活的无奈

为了拆机看无线网卡，硅脂还闹了一茬，7921太难涂了，今天刚好用完。ITX机箱又难拆，拆下换硅脂特累，返工两次。拆机清灰，一探网卡究竟，换硅脂也花了个把小时。

mac体验究极丝滑，良好，舒适，稍微安慰一下自己。

现在是晚上12:59，用的操作系统是mac os，尽管很幸苦，但是今天这么一折腾还是搞懂很多东西。

事情的原由是笔记本黑苹果操作过于卡顿，我以为是我的黑苹果配置不完善，直到一个小时前，我在笔记本重新装了（原256ssd转到台式，先128gssd），操作非常流畅，我就很好奇了？开始疑惑，也就是折腾完一天快结束的时候我才想到会不会是hidpi？因为现在的配置和之前差别最大的就是在hidpi，果然查一查，网上是通病，可怜的hd630承载了不该承受的压力，放大分辨率，显卡过载，也能理解。

翻车 - 显卡灰屏

刚好用了一周，cao，有次刚准备打危机合约的时候，刚打开mumu屏幕就灰屏，gg，所有方法都试了，基本可以确定是显卡的问题，而且是进不去系统的那种。

在b站看到有人才，如果能进系统，用讲述人功能 + 盲键盘刷回bios，那个我是真的佩服，小破站人才众多。

没办法，z390其实可以支持6代u的，但是也不能直接支持，省心还是上了9代的赛扬g4930，其实其他的都没货了而已，哎，两百六大洋呀，就为了亮机。

等了三天，东西到了，硅脂还掉了点到插槽里，把我触目惊心的，啊，这可是1200的主板呀！中途因为硅脂开不了机，搞得我慌的很，最后拿镊子纳米级工艺修复好了。

一下午，好累，这itx拆散热器巨麻烦，而且那垃圾赛扬也太慢了，我开机自启软件都能让他反应半年，最后还是大吉大利，修复成功。

达成成就：奇怪的教训增加了

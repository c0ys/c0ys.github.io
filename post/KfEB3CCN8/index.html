各平台终端代理救急

2021-08-28 1 min read

每次用都要查，还不一定查得到，在有时候突然用又忘记的时候就特别捉急呀。捉急呀。

Windows - CMD

set HTTP_PROXY=socks://127.0.0.1:7890
set HTTPS_PROXY=socks://127.0.0.1:7890

set HTTP_PROXY=socks://192.168.0.178:7890
set HTTPS_PROXY=socks://192.168.0.178:7890

set HTTP_PROXY=http://127.0.0.1:10809
set HTTPS_PROXY=https://127.0.0.1:10809

MacOS - git

设置代理

git config --global http.proxy socks5://127.0.0.1:7890
git config --global https.proxy socks5://127.0.0.1:7890

取消代理

git config --global --unset http.proxy
git config --global --unset https.proxy

MacOS - zsh

推荐clashx，贴心好用，ui现代。

export https_proxy=http://127.0.0.1:7890 http_proxy=http://127.0.0.1:7890 all_proxy=socks5://127.0.0.1:7890
